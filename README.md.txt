1) Code to represent each data structure classification and type (C language)
A. Linear Data Structures
1. Array
Elements stored in contiguous memory locations.

Direct access using index.
2. Linked List
Nodes stored in non-contiguous memory.

Each node contains data and a pointer.
3. Stack (LIFO)
Insertion and deletion from the same end (top).
4. Queue (FIFO)
Insert at rear, delete from front.
B. Non-Linear Data Structures
5. Tree (Binary Tree)
Hierarchical parent-child relationship.

6. Graph (Adjacency Matrix)
Vertices connected by edges.

Can contain cycles.
